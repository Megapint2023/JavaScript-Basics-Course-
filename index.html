Tehtävä 1

Lyhyesti, mitä koodaus tekee. 

  <button class="the-button" type="button" onclick="hitme1()"> 
  Hit me baby! </button>
  <button class="the-button" type="button" onclick="hitme2()"> 
  One more time! </button>
  
  <p id="teksti1"></p>
  <p id="teksti2"></p>
  
  <script>
    var teksti1 = "Miten menee?"
    var teksti2 = "Kuis hurisee?"
    
    function hitme1() {
     document.getElementById("teksti1").innerHTML = teksti1;
    }
    function hitme2() {
     document.getElementById("teksti2").innerHTML = teksti2;
    }
   </script>
   CSS ohje lisätty:
   .the-button {
    background-color: rgb(255, 68, 0);
    color: white;
    padding: 10px 20px;
    border: 3px solid black;
    border-radius: 5px;
    font-size: 16px;
    cursor: pointer;
  }
  .the-button:active {
    border: 1px solid black;
  }
   BREAKDOWN:
   Koodaan ensin kaksi kaksi *nappia* näkyviin. 
   Niiden alle määritän kaksi kohtaa eli ID:tä, johon funktiot syöttää
   haluamani tekstit.
   Sitten 'avaan' java scriptin ja määritän kaksi muuttujaa, joiden 
   arvona on ns. *teksti*.
   Lopuksi kaksi funktiota jotka näyttää sivostelle, määrättyyn id kohtaan
   niiden sisällön.

Tehtävä 2

Lyhyesti, mitä koodaus tekee. 

  <p id="kuvateksti1" >Which way?</p>
  <img src="images/LEFT.jpg" onclick="hitimg1()" alt="Button1" 
  style="width: 100px; height: 85px; cursor: pointer;"></img>
  
  <img src="images/RIGHT.jpg" onclick="hitimg2()" alt="Button2" 
  style="width: 100px; height: 85px; cursor: pointer;"></img>
  
  <p id="kuvateksti2" >Which way?</p>
  
  <script>
    var kuvateksti1 = "<span style='color: green'> Turn Left! </span>"
    var kuvateksti2 = "<span style='color: red'> Turn Right! </span>"
  
    function hitimg1() {
     document.getElementById("kuvateksti1").innerHTML = kuvateksti1;
    }
    function hitimg2() {
     document.getElementById("kuvateksti2").innerHTML = kuvateksti2;
    }
  
  </script>
  CSS-ohje lisätty:
  img:active {
    border: 1px solid black;
  }
  BREAKDOWN:
  Koodissa määritin kaksi ID kohtaa joihin sijoittuu/vaihtuu tekstit.
  Näiden välissä on kaksi painiketta, jotka näkyvät sivustolla kuvina.
  Eli näkyvillä on kaksi painiketta(kuvaa) joita klikkaamalla vaihtuu
  teksti sekä tekstin väri.
